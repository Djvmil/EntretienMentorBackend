Politique de confidentialité

Collecte et utilisation des informations

Votre vie privée est importante pour nous. Cette politique de confidentialité explique quelles informations nous collectons et comment nous les utilisons.

Données collectées

Nous collectons les informations suivantes :

  • Nom d'utilisateur
  • Adresse e-mail
  • Informations de localisation

Utilisation des données

Nous utilisons les données collectées pour :

  • Personnaliser l'expérience de l'utilisateur
  • Fournir un contenu personnalisé
  • Améliorer notre application

Partage des données

Nous ne partageons pas vos informations personnelles avec des tiers sauf dans les cas suivants :

  • Si cela est nécessaire pour fournir un service demandé par l'utilisateur
  • Si cela est requis par la loi

Sécurité des données

Nous prenons des mesures pour protéger vos informations personnelles contre tout accès non autorisé ou toute divulgation.

Modifications de cette politique

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page.

Pour toute question concernant cette politique de confidentialité, veuillez nous contacter à l'adresse suivante : contact@e-mentor.com
